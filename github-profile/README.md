<p align="center">
  <b>FitAndSleek</b> is a production-ready e-commerce platform built for real-world retail — from product catalog and inventory
  to checkout, <b>Bakong / KHQR</b> payments, delivery tracking, and customer support.<br/><br/>
  We ship a unified experience across <b>web storefront</b>, <b>admin dashboard</b>, and <b>Flutter mobile app</b>,
  all powered by a single <b>Laravel REST API</b>.<br/><br/>
  Powered by <b>Laravel · React · Flutter · PostgreSQL · FastAPI · CLIP / Qdrant</b>, with AI chatbot, visual image search,
  Telegram notifications, and secure auth via Sanctum + OAuth.<br/><br/>
  <i>From Cambodia to the digital storefront — we build, we ship, we scale.</i>
</p>
